# 🍽️ Zomato Restaurant Analysis using SQL

## 📌 Project Overview

This project analyzes the Zomato restaurant dataset using MySQL.

The goal of this project is to answer business questions using SQL queries and generate useful insights.

---

## 🛠 Tools Used

- MySQL
- MySQL Workbench
- GitHub

---

## 📚 SQL Concepts Covered

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- LIMIT
- COUNT()
- AVG()
- MAX()
- MIN()
- DISTINCT

---

## 📊 Business Questions Solved

- Total Restaurants
- Top 10 Restaurants by Votes
- Highest Rated Restaurants
- Online Order Analysis
- Restaurant Type Analysis
- Rating Distribution
